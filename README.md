Each key corresponds to a phrase: "GG", "Ye", "Noice"

# BOM

This is everything I need:

    3x Cherry MX Switches
    3x DSA Keycaps
    4x M3x5x4 Heatset inserts
    4x M3x16mm Screws
    1x 0.91" 128x32 OLED Display
    1x XIAO RP2040
    1x Case (2 printed parts)
    1x PCB
